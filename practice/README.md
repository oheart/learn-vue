## happyfri
- 目的： 入门vue,学习用vue构建项目； 参考项目&nbsp;[vue2-happyfri](https://github.com/bailicangdu/vue2-happyfri)
- 用vue-cli构建环境
    - npm install --global vue-cli
    - vue init webpack happyfri
    - cd happyfri
    - npm run dev
- 项目运行： 
    - 开发： npm run dev
    - 生产： npm run build
## vuex-examples
